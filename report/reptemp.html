PCs connecting to the FZM as BTSSM or other tool accessing the LMT Port must be configured with IP Addr 192.168.255.126. I have been told that IP Addr 192.168.255.130 may work as well, but I have not received test results that would verify this statement. IP Address 192.168.255.100 will no longer work with this GMC.
